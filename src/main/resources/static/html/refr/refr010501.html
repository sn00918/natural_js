Alert(N.alert)

    개요 Overview

    Alert(이하 N.alert) 는 window.alert 이나 window.confirm과 같이 메세지 대화상자를 레이어 팝업 타입으로 표현하기 위한 컴포넌트 입니다.

    Alert 대화 상자가 표시 되지 않으면 natural.config.js 파일의 N.context.attr("ui").alert.container 속성에 Alert 관련 HTML 요소들이 저장될 영역을 jQuery selector로 지정 해 주세요.

    API DEMO

    데모에는 모든 옵션 및 기능을 포함하고 있지 않습니다. 전체 기능 목록은 데모 바로 아래 있는 기능 문서들을 참고 바랍니다.

    The demo does not include all options and features. For a complete list of features, please refer to the function documentation directly below the demo.

    Default Options

    Name Type Default Required Description
    context window|jquery object null ○

    Alert 의 배경이 되는 컨텐츠를 가려주는 overlay 가 적용될 부분을 jQuery selector 를 사용하여 지정 합니다.

    window 객체를 지정하면 화면 전체를 덮어 주고 jquery object를 입력하면 지정한 요소만 덮어줍니다.

    입력요소(select, input, textarea 등)를 지정 하면 입력요소 옆에 툴팁으로 메시지를 표시 해 줍니다.
    msg (jquery object, HTML Tag string)|object undefined ○

    Alert 로 표시할 메시지

    jquery object, HTML 문자열 이나 HTML 요소 모두 인식 합니다.
    vars array undefined

    메시지의 변수를 vars로 지정한 값으로 치환 해줍니다.

    N(window).alert({
	msg : "{0} 저는 {1}입니다.",
	vars : ["안녕하세요", "홍길동"]
}).show();

/*
결과 메시지 : "안녕하세요 저는 홍길동 입니다."
*/
    html boolean false 메시지에 HTML 을 허용 할 건지 여부
    confirm boolean false true 지정 시 확인할 수 있도록 [확인], [취소] 버튼을 표시하고 false 지정 시 [취소] 버튼만 표시 합니다.
    top number undefined 메시지 대화상자의 상단(px) 위치
    left number undefined 메시지 대화상자의 좌측(px) 위치
    width number 0 메시지 대화상자의 넓이(px)
    height number 0 메시지 대화상자의 높이(px)
    title string undefined

    메시지 대화상자의 제목

    메시지 박스(context) 요소에 title 속성값이 정의 되어 있으면 정으된 값으로 지정되며 title 옵션이 undefined 이면 타이틀바를 표시하지 않습니다.
    button boolean true 버튼 표시 여부
    closeMode string remove X 버튼이나 확인 버튼을 클릭 했을때 hide 메서드를 호출 할 것인지 remove 메서드를 호출 할 것인지 지정
    - hide : hide 메서드를 호출하여 메시지 대화상자 요소를 숨겨 줌.
    - remove : remove 메서드를 호출하여 메시지 대화상자 요소를 제거 함(이전 상태가 사라짐).
    modal boolean true 모달 적용 여부
    alwaysOnTop boolean false 메시지 대화상자를 항상 최 상위에 표시 할건 지 여부
    alwaysOnTopCalcTarget string "div, span, ul, p" alwaysOnTop 옵션 적용 시 최 상위 z-index를 계산하기 위한 대상 요소지정

    jQuery selector 문자열로 지정 합니다
    dynPos boolean true 브라우저 창 크기 조절 시, 또는 부모 컨텐츠 높이가 동적으로 변경되었을때 블록 오버레이의 크기와 메시지 대화상자의 위치를 자동으로 조절 할지 여부
    draggable boolean false 메시지 창 드래그 가능 여부
    draggableOverflowCorrection boolean true 메시지 창 드래그 시 메시지 창이 화면바깥으로 넘어갔을때 자동으로 다시 안으로 넣어줄지 여부
    draggableOverflowCorrectionAddValues object {
    top : 0,
    bottom : 0,
    left : 0,
    right : 0
    }     		드래그하면서 대화상자가 window영역을 벗어날때 다시 돌아올 위치를 추가로 지정(드래그 시 횡 스크롤이 생겨 화면이 지저분 해 질때 1씩 증감하여 조절 바랍니다.)
    overlayClose boolean true 오버레이 요소를 클릭 했을 때 메세지 대화상자를 닫을지 여부
    escClose boolean true "ESC" 키를 눌렀을 때 메세지 대화상자를 닫을지 여부
    windowScrollLock boolean true modal 옵션이 true 일때 메시지 오버레이 위 에서 스크롤 했을때 윈도우 스크롤을 차단 할 것인지 여부
    saveMemory boolean false 필요없는 참조 요소를 제거하여 메모리를 절약 할 것인지 여부
    onOk function null

    확인버튼을 눌렀을때 실행되는 이벤트

    onOk : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소
}

    이벤트 핸들러 함수 에서 0을 반환 하면 대화 상자를 닫지 않습니다.
    onCancel function null

    취소 버튼을 누르거나 X 버튼을 눌렀을때 또는 메시지 오버레이를 클릭하여 메시지 대화상자가 닫혔을때 실행되는 이벤트

    onCancel : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소
}

    이벤트 핸들러 함수 에서 0을 반환 하면 대화 상자를 닫지 않습니다.
    onShow function null

    메시지 대화상자가 제거 된 후 실행되는 이벤트

    onRemove : function(msgContext, msgContents) {
	// this : N.alert 인스턴스
	// msgContext : 메시지 오버레이 요소
	// msgContents : 메시지 대화상자 요소

    함수

    Methods

    Name Arguments Type Return Description
    context - - jquery object 컴포넌트를 적용한 요소를 반환 합니다.
    selector string - selector 문자열을 입력하면 context 옵션으로 지정한 요소에서 선택된 요소를 반환 합니다.
    show - - N.alert 메시지 대화상자를 보여 줍니다.
    hide - - N.alert 메시지 대화상자를 숨깁니다.
    remove - - N.alert 메시지 대화상자를 표현하는 HTML Element들을 제거합니다.

    "ESC" 키를 눌러도 팝업이 remove 됩니다. N.alert 라이브러리를 공유하는 다른 컴포넌트(N.popup, N.grid(Validate), N.form(Validate))들도 이와 같은 동작을 합니다.

    예제 Examples

    1. 특정영역(.alertArea)에 메시지 표시

    
<div class="alertArea" style="height: 200px;border: 3px dotted #4CAF50;"></div>

<script type="text/javascript">
   	N(".alertArea").alert("It's a N.alert example").show();
</script>

    2. 전체(window) 영역에 메시지 표시

    
<script type="text/javascript">
   	N(window).alert("It's a N.alert example").show();
</script>

    3. 전체(window) 영역에 확인(Confirm) 메시지를 표시 하고 메시지 대화상자에 드래그앤드롭 기능 활성화

    
<script type="text/javascript">
   	N(window).alert({
   		msg : "It's a N.alert example",
   		confirm: true,
   		draggable: true
   	}).show();
</script>

    4. 입력요소에 메시지 표시

    
<input id="alertInput" type="text" />

<script type="text/javascript">
   	N(".alertInput").alert("It's a N.alert example").show();
</script>
